mower-plugin
============

Bienvenue sur le mower-plugin

## Pré-Requis

- JDK 7
- Maven 3+ 


## Utilisation

```java
$ mvn clean install
```

## Notes

- Ajouter un ou plusieurs fichier (.txt) dans le répertoire src/it/ressources,
- Les espaces et les sauts de lignes superflux ne sont pas tolérés, 

```
5 5
1 1 N
AAAA
1 2 N
GAGAGAGAA
3 3 E
AADAADADDA
```
